System Instructions

1. PIN Protection

The system is secured with your personal PIN.

  • Set your PIN on first access only
  • The PIN is never displayed anywhere in the system
  • Change PIN anytime using the "Change PIN" button
  • Any attempt to remove expired items will prompt for the PIN for security

2. Adding Products

  • Scan or enter the barcode, then enter quantity, description, and expiry date
  • After entering all details, press the Save Product button to store the item
  • A sound alert confirms that the product has been saved successfully

3. Automatic Sound Alerts (NEW!)

  • System automatically checks for same-day expiry items every 2 hours
  • Plays alert sound from internet if items found
  • Click the bell icon in top-right to acknowledge alerts
  • Configure alerts in "Alert Settings" section
  • Alerts help ensure no same-day expiry items are missed

4. Main Display â€“ Expiring Items

  • The main page automatically displays items expiring within the last 10 days
  • Each expiring product appears in a large, colorful bubble for easy visibility
  • The bubble includes barcode, quantity, description, and expiry date
  • This allows staff to immediately identify items that need attention

5. Expired Items

  • Expired items are automatically moved to the Expired Items section
  • Users can view all expired items in a separate list
  • To remove any expired item, the system will ask for the PIN to ensure authorized access

Usage Benefits

  • Ensures no expired items are overlooked
  • Automatic alerts for same-day expiry items
  • Helps manage inventory efficiently
  • Provides clear, visual cues for staff to act quickly
  • Secure system prevents unauthorized deletion or changes
